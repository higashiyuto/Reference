<option>タグの詳細

    value
    ・選択肢が選ばれた時に送信されるデータ

    ```html
    <option value="apple">りんご</option>

    selected
    ・初期選択状態にする

    ```html
    <option value="grape" selected>ブドウ</option>
    ```
    -> 初期状態で「ブドウ」が選択される

    disabled
    ・選択できない状態にする

    ```html
    <option value="banana" disabled>バナナ</option>
    ```
    -> バナナは表示されるが選択不可
